# Climate Events Dashboard

An analytical dashboard developed to explore historical precipitation patterns and extreme climate events in Southern Brazil between 2023 and 2025.

The project was built using Python, Pandas and Power BI, combining data extraction, transformation, modeling and visual storytelling to generate analytical insights about seasonality, regional criticality and precipitation behavior.

## Project Objective

This project aims to analyze:

* precipitation accumulation patterns
* seasonality behavior
* recurrence of extreme weather events
* regional climate criticality

The analysis was designed to answer questions such as:

* Which state historically recorded the most extreme events?
* Which periods showed the highest precipitation accumulation?
* Are there identifiable temporal patterns related to severe climate events?

## Technologies Used

* Python
* Pandas
* Power BI
* DAX
* Open Meteo API

## Data Pipeline

Open Meteo API → Python → Pandas → CSV → Power BI

## Main Insights

* Rio Grande do Sul showed the highest recurrence of extreme precipitation events.
* Severe events were concentrated mainly in specific seasonal periods.
* Distinct precipitation behaviors were identified between Southern Brazilian states.
